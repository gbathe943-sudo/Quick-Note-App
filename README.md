# Quick Note Application

A full-stack web application built using Flask, SQLite, HTML, CSS, and JavaScript.

## Features
- Create Notes
- View Notes
- Search Notes
- Delete Notes
- Dark Mode
- SQLite Database Storage

## Technologies Used
- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript

## Run Locally

pip install -r requirements.txt

python database.py

python app.py
